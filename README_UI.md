# RideLane UI設計
バージョン: 1.0.39  
更新日: 2026-02-15

## 1. 画面構成
- Home: 地図、天気カード、ログダッシュボード、ナビ開始ボタン
- Destination: 100km 圏内の目的地検索
- Route Preview: ルート全体確認 + スタート/キャンセル
- Driving: 進行方向追従、案内カード、ズーム操作、終了/再設定

## 2. デザイン方針
- ライトテーマ（Citrus Road）
- 高コントラストな文字色
- 天気カードは条件連動グラデーションのリッチ表示（ウィジェット風）

## 3. 表示ルール
- 単位は `km` / `km/h`
- 天気の風速単位は `m/s`
- Home 右下に `verX.Y.Z` を表示
- 目的地候補は現在地 100km 圏内に限定

## 4. 操作ルール
- ルート開始前に必ずプレビューを表示
- ナビ中ズームUIは右下固定
- ルート取得失敗時は警告メッセージを表示
